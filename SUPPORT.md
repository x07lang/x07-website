# Support

This repository hosts the X07 public website and documentation site (https://x07lang.org/).

## Questions

If you have questions about using X07 (toolchain, packages, policies), use:

- Canonical docs: https://x07lang.org/docs/
- GitHub Discussions (recommended): https://github.com/x07lang/x07/discussions

## Bug reports

GitHub Issues in this repo are for actionable website build/deployment bugs, doc rendering bugs, and sync pipeline bugs.

For docs content issues that originate from the toolchain repo source docs, file the issue in https://github.com/x07lang/x07 and link to the broken page.

## Security issues

Do not open public issues for security problems.

See [SECURITY.md](SECURITY.md).
