# service_workflow_service_v1

Starter scaffold for the `svc_workflow_service_v1` service archetype.

## Purpose

Minimal workflow-service scaffold for orchestration state machines. This archetype is scaffold-only until workload packaging and runtime orchestration land end to end.

## Included surfaces

- `arch/service/index.x07service.json`
- `arch/manifest.x07arch.json`
- `src/main.x07.json`
- `src/example.x07.json`
- `tests/tests.json`

This example is intentionally minimal.
It exists to seed `x07 init` and later service tooling.
